1. Reminder App

Author: Di Gao | Supervisor: Armaan Dhanji


A browser based web application to allow users remind - Daily Todo Tasks, Meetings, Homework and Assignments, Business Appointments, Taking Medication/Pills, Paying Bills, Policy renewals, Important Calls, Birthdays, Anniversaries and many more. Also users are able to share those reminders with other users of the application.

Technologies:

  • HTML, CSS
  • JavaScript
  • JQuery
  • Bootstrap
  • Heroku

Features:

  • Safe login/logout function
  • Personal data storage solution
  • Easy to maintaion for admin

A version control tool(Github) also be used in this project for tracking and managing changes to code.
